This project is part of the course SenseBox for People-Centered Urban Planning at the Institute for Geoinformatics, MÃ¼nster, which took place in winter term 2015/16.


SenseBox is a hardware and software platform designed to empower citizens by providing them easy access to (geo)data-gathering and (geo)data-processing. The arduino-based hardware can be easily customised to include a variety of sensors. In the typical use-case, a pre-assembled SenseBox is sent directly to the final user who customises simple computer code and starts contributing geo-tagged data to the community.

This application focuses on bicycling roads, which are presented on a map with perfect or bad conditions. A user can measure cycling tracks, which are recorded on a SenseBox and his bike. After the collection, the data can be easly uploaded and shared with the community.

If you want to test this application, please follow the instructions, which can be found on GitHub. The Arduino-Sketch for the SenseBox and all requirements can also found there: https://github.com/mrunde/SenseBox-PerfectCyclingRoads
